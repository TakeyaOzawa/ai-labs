# Tech Blog Material Scout — 収集対象ソース参照

## 公式ドキュメント・リファレンス
各技術の公式ドキュメントサイト, GitHub リポジトリ (README/CHANGELOG/Discussions), RFC/PEP/TC39 Proposals

## 技術ブログ・チュートリアル（深掘り用）
公式ブログ詳細記事, 著名エンジニアの技術ブログ, カンファレンス発表資料・動画

## ベンチマーク・比較記事
公式ベンチマーク結果, 第三者による比較検証記事（複数ソースで裏どり）

## コミュニティ反応
Hacker News (news.ycombinator.com), Reddit (r/programming等), X (Twitter), Zenn, Qiita

## 流行動向調査
Qiita (タグ別記事数, LGTM数, ストック数, トレンド), Zenn (トピック別記事数, Like数, Book有無, スクラップ活発度, トレンド)

## 検索クエリパターン
- 公式情報: `{技術名} official documentation` / `{技術名} release notes {バージョン}` / `{技術名} changelog`
- 技術詳細: `{技術名} architecture design` / `{技術名} benchmark performance` / `{技術名} vs {競合} comparison`
- コミュニティ: `{技術名} site:news.ycombinator.com` / `{技術名} site:reddit.com`
- 既存記事: `{技術名} site:zenn.dev` / `{技術名} site:qiita.com` / `{技術名} ブログ 解説`
- 流行動向: `{技術名} Qiita トレンド` / `{技術名} Zenn トレンド`
